    背景

    我们知道，在使用Hexo写博客的时候需要手动创建md文件,并且需要将文件上传到github上，如果说这些操作不用命令那可能会比较麻烦，所以这里跟大家介绍几个实用的Hexo命令来帮你快速写文章以及上传文章等等，这些操作统统打包带走!

    命令大全

    $ hexo clean                清理public的内容（ps:清理之前需注意图片保存）。
$ hexo g                    生成静态页面至public目录
$ hexo d                    将public目录部署到GitHub上
$ hexo help                 查看帮助
$ hexo version              查看Hexo的版本
$ hexo s                    本地启动服务（一般写作时用得到）
$ hexo new '文章名字'       新建文章(一般都用这个创建文章哦！)
$ hexo new page '页面名字'  新建页面（此页面不会作为文章出现在博文目录哦！）

    组合命令

    $ hexo s -g  #生成静态页面至public并启动本地服务
$ hexp d -g  #生成静态页面至public并上传道GitHub(这个我是比较常用的)
