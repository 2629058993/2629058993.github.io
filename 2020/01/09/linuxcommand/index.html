    命令大全

    $ ps -ef |grep redis(可更换)  查看redis进程可查看任意进程

$ rm -f  文件名           强制删除

$ pwd      显示目前的目录

$ mkdir：创建一个新的目录

$ rmdir：删除一个空的目录

$ cp: 复制文件或目录

$ rm: 移除文件或目录

$ mv: 移动文件与目录，或修改文件与目录的名称

$ pkill redis     强制结束redis进程

$ kill -9  pid       根据pid来结束指定进程

$ make && make install  进行编译

$ tar -zxvf  压缩包   进行解压

$ gcc -v  查看gcc是否安装 （c环境）

$ systemctl status firewalld     查看防火墙状态

$ systemctl stop firewalld    关闭防火墙

$ wget http://erlang.org/download/otp_src_18.2.1.tar.gz  wget下载指定的url地址

$ export 设置环境变量或者显示环境变量 。  export  -p  列出当前的环境变量值

$ source /etc/profile    使修改/etc/profile配置生效

$ firewall-cmd --zone=public --add-port=15672/tcp --permanent   防火墙放开端口
    firewall-cmd --reload  重启防火墙

$ su es   ---切换es用户。

$ 创建用户组和用户（elasticsearch不能用root用户启动）
groupadd elasticsearch # 创建elasticsearch组 

$ elasticsearch组指定到es用户 并设置用户es的密码为es
useradd es -g elasticsearch -p es

$ 给用户es授权可操作的目录
chown -R es:elasticsearch /usr/local/es

$ sudo chmod 777 -R /usr/local/es/  ------将当前目录下所有文件及目录和目录下的文件权限修改成可读可写可执行,最高权限
